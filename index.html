WordLenz — Privacy Policy

Last updated: 2026-06-07
English한국어

English

WordLenz does not collect, store, or transmit your personal data to us. There is no server and no user account. Everything you create stays on your device.

What stays on your device

  • Saved words, books, and review schedules are stored locally in an on-device database (SQLite).
  • API keys are stored in the device's secure storage (iOS Keychain / Android Keystore). They never leave your device except when a request is sent directly to the AI provider you chose.
  • Photos you capture are processed on-device and stored locally as part of your entries.

Camera & Photos

  • Camera is used only to photograph highlighted/underlined text you want to analyze.
  • Photo library access is used only when you pick an existing image to analyze.

We never upload your photos to us.

AI providers (important)

To interpret a word or sentence, the image or text is sent directly from your device to the AI provider you choose and configure (Google Gemini, OpenAI, or Anthropic) using your own API key. We do not proxy, see, or store this data. Your use of these services is governed by their respective privacy policies:

  • Google Gemini — ai.google.dev/gemini-api/terms
  • OpenAI — openai.com/policies/privacy-policy
  • Anthropic — anthropic.com/legal/privacy

Backup

When you explicitly use Export Data, WordLenz creates a single backup file and lets you save it through the system share sheet (Files, iCloud, Google Drive, etc.). This file contains your API keys in plain text — keep it private. We never receive this file.

No tracking

WordLenz contains no analytics, no advertising, and no third-party tracking SDKs.

Children

WordLenz is not directed to children under 13.

Contact

Questions: hyunsoo.go@gmail.com

한국어

WordLenz는 사용자의 개인정보를 수집·저장하거나 당사로 전송하지 않습니다. 별도의 서버나 계정이 없으며, 사용자가 만든 모든 데이터는 기기 안에만 저장됩니다.

기기에만 저장되는 데이터

  • 저장한 단어·책·복습 일정은 기기 내 데이터베이스(SQLite)에 로컬 저장됩니다.
  • API 키는 기기 보안 저장소(iOS Keychain / Android Keystore)에 저장되며, 사용자가 선택한 AI 제공자에게 직접 요청을 보낼 때를 제외하고 기기를 벗어나지 않습니다.
  • 촬영한 사진은 기기에서 처리되어 항목의 일부로 로컬 저장됩니다.

카메라 · 사진

  • 카메라는 해석할 형광펜·밑줄 표시 텍스트를 촬영하는 데만 사용됩니다.
  • 사진 보관함은 해석할 기존 이미지를 선택할 때만 접근합니다.

당사는 사진을 수집하지 않습니다.

AI 제공자 (중요)

단어·문장을 해석하기 위해, 이미지 또는 텍스트는 사용자가 직접 선택·설정한 AI 제공자(Google Gemini, OpenAI, Anthropic)에게 사용자 본인의 API 키로 기기에서 직접 전송됩니다. 당사는 이 데이터를 중계·열람·저장하지 않습니다. 해당 서비스 이용에는 각 사의 개인정보 정책이 적용됩니다.

백업

데이터 내보내기를 실행하면 단일 백업 파일이 생성되어 시스템 공유 시트(파일, iCloud, Google Drive 등)로 저장할 수 있습니다. 이 파일에는 API 키가 평문으로 포함되므로 안전하게 보관하세요. 당사는 이 파일을 수신하지 않습니다.

추적 없음

WordLenz에는 분석 도구·광고·제3자 추적 SDK가 전혀 없습니다.

아동

WordLenz는 만 13세 미만 아동을 대상으로 하지 않습니다.
